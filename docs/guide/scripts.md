# 脚本管理

脚本管理提供了玄武面板的在线文件资源浏览器和代码编辑器，用于管理 `scripts` 目录下的所有脚本文件。

## 资源管理器

### 目录树

以树形结构展示 `scripts` 目录下的所有文件和文件夹，目录优先排序，同类型按名称字母排序。自动过滤 `__pycache__` 等缓存目录。

### 文件操作

| 操作 | 说明 |
| :--- | :--- |
| **文件预览** | 点击文件可查看其文本内容 |
| **在线编辑** | 在编辑器中修改文件内容后保存到服务器 |
| **创建文件/目录** | 在指定路径下创建新的文件或文件夹 |
| **重命名** | 重命名文件或文件夹（禁止跨目录重命名） |
| **移动** | 将文件或文件夹移动到目标路径 |
| **复制** | 复制文件到目标路径 |
| **删除** | 删除指定的文件或文件夹 |
| **下载** | 下载文件到本地 |

### 上传功能

- **压缩包上传**：支持 `.zip`、`.tar`、`.gz`、`.tgz` 格式的压缩包，上传后自动解压到目标目录
- **多文件上传**：支持多选文件上传，保持原始目录结构

## 在线编辑器

### 语法高亮

集成现代代码编辑器，支持 JavaScript、Python、Shell、Go、TypeScript、JSON 等主流开发语言的语法高亮显示。

### 编辑增强

- **查找与替换**：支持文本搜索和批量替换
- **自动缩进**：代码自动格式化
- **括号匹配**：自动匹配并高亮对应括号
- **未保存提示**：内容修改后离开页面时弹出确认对话框

### 一键保存

编辑完成后，内容实时写入服务器端物理磁盘。配合定时任务可快速验证脚本修改效果。

### 在线运行

编辑器支持直接运行当前脚本，可选择指定的运行环境（Mise 管理的语言版本），并在内置终端中查看实时输出。

## 路径安全

所有文件操作均限制在 `scripts` 目录内，系统通过路径校验机制防止目录穿越攻击，确保无法访问系统敏感文件。

路径校验规则：
- 所有路径经过 `filepath.Clean` 清理
- 使用相对路径计算确保不超出 `scripts` 根目录
- 拒绝包含 `..` 的路径穿越尝试
