# 数据仪表

数据仪表是玄武面板的首页，提供面板运行状态的全局概览、任务执行统计及关键指标的可视化展示。

## 总体概览

仪表盘顶部展示六项核心指标卡片，每项指标均可点击直接跳转到对应页面：

| 指标 | 说明 | 跳转目标 |
| :--- | :--- | :--- |
| **任务总数** | 系统中已创建的定时任务数量 | 任务管理 |
| **今日执行** | 今天所有任务的累计执行次数 | 执行历史 |
| **环境变量** | 已配置的环境变量总数 | 变量机密 |
| **日志总数** | 历史执行日志记录总数 | 执行历史 |
| **调度注册** | 已注册到后台 Cron 调度器的任务数 | 任务管理 |
| **正在运行** | 当前正在执行中的任务数量 | 执行历史（running 筛选） |

指标卡片支持数字滚动动画，数据通过 WebSocket 实时刷新。

## 执行状况统计

### 每日执行趋势图

通过折线图展示近 15 天（移动端）或 30 天（桌面端）内每天的任务执行情况：

- **成功**：按天聚合所有 `success` 状态的日志数量
- **失败**：按天聚合所有 `failed` 状态的日志数量

缺失日期自动填充为零值，确保图表连续。可通过切换时间范围查看不同周期的统计数据。

### 任务执行占比

饼图展示各任务的执行次数占比，帮助识别高频任务和异常任务。数据同样支持 15 天和 30 天的时间范围切换。

## 每日一言

仪表盘底部展示随机古诗词（每日一言），为技术面板增添文化气息。数据来源于内置的古诗词库，每次加载随机获取。

## API 接口

仪表盘数据通过以下接口获取：

| 接口 | 方法 | 说明 |
| :--- | :--- | :--- |
| `/api/v1/stats` | GET | 获取六项核心统计指标 |
| `/api/v1/sendstats` | GET | 获取每日执行趋势（支持 `days` 参数，1-90） |
| `/api/v1/taskstats` | GET | 获取任务执行占比统计（支持 `days` 参数，1-90） |
| `/api/v1/sentence` | GET | 获取随机古诗词 |
