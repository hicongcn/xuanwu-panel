# 快速开始

## 访问面板

部署完成并启动服务后，打开浏览器访问：

```
http://localhost:8052
```

（如果您修改了端口或使用反向代理，请替换为实际地址）

## 默认账号

| 项目 | 值 |
| :--- | :--- |
| 用户名 | `admin` |
| 密码 | 12 位随机密码，见下方获取方式 |

### 获取初始密码

首次启动时，系统会自动生成 12 位随机管理员密码，并打印在启动日志中。

**Docker 部署**：

```bash
docker logs xuanwu | grep "密码"
```

**二进制部署**：

查看终端输出中包含「密码」或 `password` 的日志行。

::: warning 重要
请在首次登录后立即修改管理员密码，确保系统安全。
:::

## 首次使用建议

登录面板后，建议按以下顺序完成初始配置：

### 1. 修改管理员密码

进入右上角用户头像下拉菜单 → 「个人设置」，修改管理员密码。

### 2. 配置调度设置

进入「系统设置」→「调度设置」，根据服务器资源调整：

| 配置项 | 默认值 | 说明 |
| :--- | :--- | :--- |
| 并发数 | 4 | 同时执行的最大任务数 |
| 最大队列大小 | 100 | 等待执行的任务队列上限 |

### 3. 安装语言依赖

进入「编程语言」页面，根据需要安装脚本运行所需的语言环境（如 Python、Node.js 等）。通过 Mise 管理，支持多版本并存，可按需安装。

### 4. 配置消息推送（可选）

进入「消息推送」页面，根据需求配置通知渠道（如企业微信、Telegram、Bark 等），用于接收任务执行结果通知。

## 功能一览

| 模块 | 说明 |
| :--- | :--- |
| **仪表盘** | 实时监控任务执行动态、系统资源状态 |
| **定时任务** | 创建和管理 Cron 定时任务，支持并发控制、超时管理 |
| **脚本管理** | 在线代码编辑器，支持文件上传、压缩包解压、目录导航 |
| **在线终端** | WebSocket 实时终端，浏览器中直接执行命令 |
| **编程语言** | Mise 运行时管理，一键安装 20+ 编程语言 |
| **依赖管理** | 各语言第三方库的可视化安装与管理 |
| **环境变量** | 安全存储敏感配置，任务运行时自动注入 |
| **消息推送** | 十余种推送渠道配置与任务通知绑定 |
| **仓库同步** | 从 GitHub 仓库自动同步脚本 |
| **消息日志** | 推送日志、登录日志、调度日志、系统事件查看 |
| **系统设置** | 站点配置、调度参数、备份恢复、账号安全 |
