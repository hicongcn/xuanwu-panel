# 定时任务

定时任务是玄武面板的核心模块，支持对各类多语言脚本、命令进行精细化的 Cron 调度管理。

## 任务属性

创建或编辑任务时，可配置以下属性：

| 属性 | 说明 | 示例 |
| :--- | :--- | :--- |
| **名称** | 任务的直观标识 | `每日签到` |
| **备注** | 任务的补充说明 | `自动执行京东签到脚本` |
| **Cron 表达式** | 标准 cron 规则（分 时 日 月 周） | `0 8 * * *` |
| **命令** | 要执行的脚本路径或 Shell 命令 | `python3 sign.py` |
| **前置命令** | 任务执行前运行的命令 | `pip install requests` |
| **后置命令** | 任务执行后运行的命令 | `rm -f /tmp/cache` |
| **工作目录** | 命令执行的工作目录，为空则使用 `scripts` 目录 | `$SCRIPTS_DIR$/repo1` |
| **超时时间** | 单次运行的最大时长（分钟），默认 30 分钟 | `60` |
| **标签** | 逗号分隔的分类标签，用于筛选和批量管理 | `签到,日常` |
| **失败重试** | 失败后的重试次数和间隔（秒） | 重试 3 次，间隔 60 秒 |
| **随机延迟** | 执行前随机延迟的秒数范围 | `300`（0~300 秒随机延迟） |
| **语言配置** | 指定任务使用的语言运行时版本 | `python@3.11` |
| **环境变量** | 关联的环境变量 ID 列表 | 选中多个变量 |
| **置顶类型** | 任务是否置顶显示 | `top` / `none` |

## 任务类型

### 普通任务 (`task`)

标准的脚本定时执行任务，需要指定命令和 Cron 表达式。

### 仓库同步任务 (`repo`)

通过 Git 仓库同步脚本并自动注册任务，配置项通过 JSON 格式存储在 `config` 字段中，包含仓库地址、分支、白名单/黑名单等。详见 [仓库同步](sync.md)。

## 触发类型

### Cron 定时触发

标准的定时调度，根据 Cron 表达式在指定时间自动执行。

### 开机启动 (`xuanwu_startup`)

当触发类型设置为 `xuanwu_startup` 时，该任务将在面板主进程启动后立即执行一次。

适用场景：
- 自动挂载磁盘或网络共享
- 环境预热（如安装 PHP 编译依赖）
- 启动自定义的后台常驻服务

## 管理操作

### 启停控制

通过切换开关即可启用或禁用任务。禁用后任务将从后台 Cron 调度队列中摘除，不会自动执行。

### 立即执行

不等待 Cron 触发时间，即刻拉起任务运行。支持通过面板按钮或命令行 `xuanwu task run` 下发。

### 停止运行

对正在执行中的任务，可以通过日志 ID 终止其运行进程。

### 查看日志

直接跳转到该任务关联的最新执行历史详情，支持实时查看运行中任务的输出。

### 批量操作

支持以下批量操作方式：
- **勾选批量删除**：选中多个任务后一键删除
- **按条件批量删除**：根据名称、标签、类型等条件批量删除匹配的任务

## Cron 表达式预设

编辑任务时，提供常用的 Cron 预设选项，方便快速选择：
- 每分钟执行
- 每小时整点
- 每天零点
- 每周一
- 自定义表达式

## 仓库任务去重

仓库同步任务通过 `source_url` + `branch` 生成唯一标识（SourceID），系统自动检测重复。如果已存在相同标识的任务，创建操作将自动转为更新操作。

## API 接口

| 接口 | 方法 | 说明 |
| :--- | :--- | :--- |
| `/api/v1/tasks` | GET | 分页获取任务列表（支持 name/tags/type 筛选） |
| `/api/v1/tasks` | POST | 创建任务 |
| `/api/v1/tasks/:id` | GET | 获取任务详情 |
| `/api/v1/tasks/:id` | PUT | 更新任务 |
| `/api/v1/tasks/:id` | DELETE | 删除任务（支持 `delete_files=true` 删除仓库文件） |
| `/api/v1/tasks/batch-delete` | POST | 按 ID 批量删除 |
| `/api/v1/tasks/batch-by-query` | DELETE | 按查询条件批量删除 |
| `/api/v1/tasks/stop/:logID` | POST | 停止运行中的任务 |
| `/api/v1/tasks/tags` | GET | 获取所有任务标签 |
| `/api/v1/execute/task/:id` | POST | 立即执行任务 |
