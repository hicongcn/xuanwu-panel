# 脚本示例

玄武面板内置了一批可直接参考的脚本示例，适合用来验证运行环境、演示常见用法，或作为自定义脚本的起点。

## 示例同步

使用 Docker 镜像启动时，容器启动后会自动将仓库中的 `example` 目录同步到脚本目录 `data/scripts/example/` 下。

## 使用前准备

1. 确认示例文件已同步到脚本目录。
2. 根据脚本语言，到「语言依赖」页面安装对应依赖包。
3. 按实际环境修改脚本中的地址、Token 等配置。
4. 在「定时任务」中创建任务并关联对应脚本运行。

::: tip
如果示例脚本依赖第三方包但尚未安装，任务会报缺少模块或包的错误。请先在「语言依赖」页面完成安装。
:::
