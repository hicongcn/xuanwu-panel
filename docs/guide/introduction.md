# 项目介绍

**玄武面板 (Xuanwu Panel)** 是一款极致轻量、高性能的自动化任务调度平台。采用 Go + Vue3 架构，深度集成 Mise 运行时管理，支持几乎所有主流编程语言的动态安装与统一依赖管理。

- **GitHub**：[https://github.com/hicongcn/xuanwu-panel](https://github.com/hicongcn/xuanwu-panel)
- **Docker 镜像**：`ghcr.io/hicongcn/xuanwu:latest`

## 技术栈

| 层级 | 技术 | 说明 |
| :--- | :--- | :--- |
| 后端 | **Go (Gin)** | 高并发、低内存占用的 HTTP 服务 |
| 前端 | **Vue3 (Vite)** | 现代化响应式 UI，支持 PWA |
| 数据库 | **SQLite / MySQL** | 默认嵌入式 SQLite，可切换 MySQL |
| 运行时 | **Mise** | 动态版本管理，20+ 语言一键安装 |
| UI 组件 | **Shadcn/ui + TailwindCSS** | 深色/浅色主题，移动端适配 |

## 核心特色

### 多语言支持 (20+)

深度集成 Mise 运行时管理器，支持 Python、Node.js、Go、Rust、PHP、Java、Ruby、Deno、Bun、Lua、Perl、Elixir、Dart、Nim、Crystal、.NET 等 20+ 编程语言的动态安装、多版本并存和统一依赖管理。

### 统一依赖管理

通过内置依赖管理器，每种语言拥有专属的依赖安装方案，支持在面板中一键安装第三方库，无需手动进入终端操作。

### 任务调度

- 标准 Cron 表达式调度，常用时间规则快捷选择
- 支持并发队列控制与最大并发数配置
- 任务超时管理与自动终止
- 开机启动任务支持
- 任务通知绑定，执行结果自动推送

### 脚本管理

- 基于 Monaco Editor 的在线代码编辑器，支持语法高亮
- 文件上传与压缩包解压
- 目录树导航，多文件管理
- 运行配置一键切换

### 在线终端

WebSocket 实时终端，直接在浏览器中执行命令，支持 ANSI 颜色输出，适合运维调试。

### 消息推送

内置十余种主流推送渠道：

| 类型 | 渠道 |
| :--- | :--- |
| 国内即时通讯 | 企业微信、钉钉、飞书 |
| 国际即时通讯 | Telegram |
| 推送服务 | Bark、PushPlus、PushMe |
| 自建服务 | Gotify、Ntfy、Vocechat、WxPusher |
| 邮件 | SMTP 邮件 |
| 企业通讯 | Dtalk |
| 自定义 Webhook | 支持任意 HTTP 接口 |

### 安全特性

- AES-GCM 加密存储敏感配置（环境变量、通知密钥）
- 登录防暴力破解
- 路径遍历攻击防护
- 审计日志记录

### 仓库同步

支持从 GitHub 仓库同步脚本，自动解析仓库结构并导入脚本文件。

### PWA 支持

前端支持 PWA（渐进式 Web 应用），可安装到桌面或移动设备，提供类原生应用体验。

### 其他特性

- **仪表盘**：实时监控任务执行动态与系统状态
- **环境变量管理**：安全存储与任务运行时自动注入
- **备份恢复**：支持面板数据一键备份与恢复
- **CLI 工具**：支持命令行重置密码、数据恢复等操作
- **日志查看**：登录日志、推送日志、调度日志、系统事件分栏查看
- **移动端适配**：响应式设计，小屏设备也能流畅使用
