# 终端命令

终端模块允许用户在 Web 端直接与服务器的 Shell 环境进行交互式操作。

## WebSocket 双工连接

终端通过 WebSocket 协议建立全双工连接，支持实时的命令输入和输出回传。

### PTY 伪终端（Unix/macOS）

在 Unix 和 macOS 系统上，终端使用 PTY（伪终端）模式：
- 支持完整的终端交互，包括 `yes/no` 确认、`npm init` 交互式问答等
- 默认终端窗口大小为 24 行 × 80 列
- 支持动态调整终端窗口大小（客户端发送 resize 消息）
- 设置 `TERM=xterm-256color` 环境变量

### Pipe 模式（Windows）

在 Windows 系统上，终端使用标准的 stdin/stdout/stderr Pipe 模式：
- 不支持终端窗口大小调整
- 同时读取 stdout 和 stderr 输出

## 实时输出回传

- 命令执行结果通过 WebSocket 实时推送到浏览器
- 支持 ANSI 转义序列，正确渲染终端彩色文本和样式
- 自动处理 GBK 编码到 UTF-8 的转换，兼容中文 Windows 环境
- 心跳机制（Ping/Pong）确保连接稳定性，超时 60 秒自动断开

## 工作目录

终端默认在 `scripts` 目录下启动，用户可以直接在该目录中操作脚本文件。

## 环境注入

终端会自动注入以下环境：
- **用户环境变量**：在变量机密页面配置的所有环境变量
- **Mise 运行时路径**：已安装的 Node.js 全局依赖路径（`NODE_PATH`）
- **CLI 工具路径**：`xuanwu` 命令行工具的运行时路径
- **数据库配置**：确保终端中执行的子命令能正确连接数据库

## 可用命令

终端页面顶部提供常用命令的快速参考，包括：
- `reposync` - 同步远程 Git 仓库或文件到本地
- `resetpwd` - 重置管理员密码
- `restore` - 从备份恢复系统数据
- `builtininstall` - 安装内建助手库
- `task` - 任务管理（列表/触发/启停/状态/历史）

支持一键复制命令到剪贴板。

## 安全限制

- 演示模式下终端功能被禁用，显示提示信息
- WebSocket 连接需要管理员权限认证
- 最大消息大小限制为 1MB
